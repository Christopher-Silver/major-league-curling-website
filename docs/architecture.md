# Website Architecture

The Major League of Curling website is built using a lightweight CMS-based architecture.

## Components

- WordPress (core platform)
- Elementor (page layout and design)
- SportsPress (league data: teams, standings, schedules)

## Content Flow

1. League results are collected weekly
2. Standings and schedules are updated manually
3. Pages are updated through WordPress dashboard
4. Social media posts are created to promote updates

## Constraints

- No paid plugins or custom backend
- Manual updates required for accuracy
- Limited automation due to free-tier tooling

## Future Improvements

- Automated standings ingestion
- Database-backed results
- Admin dashboard for score entry
- API-based data updates
